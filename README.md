# 👨‍💻 Sachintha Navindu Fernando - Portfolio

A modern, responsive portfolio website built with React and Tailwind CSS. Features a sleek black and white theme with smooth animations and a professional layout.

## 🚀 Live Demo

**[View Live Portfolio](https://portfolio-2-61bcc.web.app)**

## ✨ Features

- 🎯 **Single Page Portfolio** - Smooth scrolling between sections
- 📱 **Fully Responsive** - Optimized for all devices (mobile, tablet, desktop)
- 🎨 **Modern Design** - Black and white theme with gradient accents
- ⌨️ **Typing Animation** - Dynamic text animation in the about section
- 🧭 **Navigation Menu** - Fixed navbar with active section highlighting
- 📊 **Skills Section** - Visual representation of technical skills with icons
- 📚 **Education Timeline** - Clean timeline layout for education history
- 💼 **Projects Showcase** - Card-based project display with tech stack tags
- 🔗 **Social Links** - LinkedIn, GitHub integration
- 🎭 **Smooth Animations** - Hover effects, transitions, and scroll animations

## 🛠️ Technologies Used

- **Frontend Framework:** React.js
- **Styling:** Tailwind CSS
- **Icons:** React Icons
- **Animations:** CSS Animations
- **Build Tool:** Vite
- **Deployment:** Firebase Hosting
